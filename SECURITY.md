# Security Policy

Pigeon is an early-stage prototype and should **not** be used to protect production
payment or PCI data as-is. That said, security reports are taken seriously.

## Supported versions

The project is pre-1.0. Only the latest `main` receives fixes.

| Version | Supported |
|---------|-----------|
| `main`  | ✅        |
| < 0.1   | ❌        |

## Reporting a vulnerability

Please report vulnerabilities **privately**. Do not open a public GitHub issue.

- Preferred: open a [private security advisory](https://github.com/vishnu-77/pigeon/security/advisories/new)
  on this repository.
- Alternatively, contact the maintainer through the email on the
  [vishnu-77 GitHub profile](https://github.com/vishnu-77).

Please include:

- a description of the issue and its impact,
- steps to reproduce (a minimal message/policy that triggers it),
- affected files or endpoints.

You can expect an acknowledgement within a few days. Once a fix is available, we will
coordinate disclosure.
